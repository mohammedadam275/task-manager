# Task Management Web App

## Author
Mohammed Adam

## Overview
A modern task management web application built with React. Users can create, update, delete, and organize tasks using filters, categories, and preset task bundles. The app focuses on clean UI, usability, and core CRUD functionality.

## Features
- Create, read, update, and delete todos
- Mark tasks as completed
- Filter todos by all, completed, or incomplete
- Category-based preset task bundles (Health, Travel, Work, School, Personal, Errands)
- Controlled contact form
- Dark mode toggle
- Modern, responsive user interface

## How to Use
- Navigate to the Todos page to add and manage tasks
- Use filters to view specific task states
- Use Categories to quickly add preset tasks
- Navigate to the Contact page to submit contact information

## Technologies Used
- React
- JavaScript
- HTML
- CSS
